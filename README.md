🏎️ F1 Chatbox

A conversational AI-powered chatbot built around Formula 1 — designed to answer questions, provide insights, and make exploring F1 more interactive and fun.

Whether you're a casual fan or someone who tracks every race weekend, this project lets you “talk” F1 instead of just reading about it.

🚀 What this project does

F1 Chatbox is a simple but powerful chatbot that lets users:

Ask questions about Formula 1
Get quick, contextual answers about drivers, teams, races, and history
Interact with F1 data in a conversational way
Explore the sport without digging through multiple sources

Think of it as a lightweight F1 assistant.

🧠 Why I built this

Most F1 content is either:

Static (articles, stats pages), or
Scattered across different platforms

I wanted to build something more interactive and intuitive — where you can just ask:

“Who won the 2021 championship?”
“Tell me about Ferrari’s performance this season”

…and get answers instantly.

This project is also part of my exploration into:

AI-powered interfaces
Chat-based UX
Applying LLMs to niche domains

🏗️ How it works
User enters a question in the chat interface
The input is processed and sent to the backend
The backend queries the AI model
The model generates a response based on context
The response is displayed in the chat UI

Simple flow, but powerful experience.
